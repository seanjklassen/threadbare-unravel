quick start

unravel is installed. open your DAW and rescan plugins if needed. then:

  1. insert unravel on a track or aux return in your DAW.
  2. select a preset in the upper-left to set a starting point.
  3. drag the puck to shape the reverb character in real time.
  4. need more control? open the lower-right settings for deeper parameter adjustments.
  5. explore disintegration loop mode to capture, degrade, and recycle ambience for evolving textures.
